## ⚙개발 환경

- IOS 13.0+
- Xcode 12.5

<br>

## 📚라이브러리 및 API

| 라이브러리 및 API | 목적 |
| --- | --- |
| Firebase | SMS 인증 |
| Alamofire | 서버 통신 |
| Kingfisher | 이미지 불러오기 |
| SnapKit | AutoLayout |
| XLPagerTabStrip | 상단 탭바 |
| Kakao Map, 키워드로 장소검색 API | 지도 사용 |
